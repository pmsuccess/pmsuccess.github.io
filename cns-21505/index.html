全球唯二！台灣政府頒布CNS 21505(專案治理指引國家標準) 有望大幅改善企業專案能力 

PMI-Taipei Taiwan Chapter（PMI-TW）為促進台灣專案管理發展，因企業缺乏公定標準可依循，於2016年成功將國際標準 ISO 21500 轉化為國家標準 CNS 21500。隨後，PMI-TW 進一步號召志工，將 ISO 21504 和 ISO 21505 轉化為 CNS 21504 和 CNS 21505，並在2018年持續推動 ISO 21503、ISO 21508、ISO 21511 的轉化。此系列轉化旨在加速台灣專案管理接軌國際，健全企業執行專案時的標準化依循，提升台灣專案管理生態。

  • 周龍鴻
  • 02/27/2025

在2016年成功將第一個關於專案管理的國際標準 ISO 21500轉化為國家標準 CNS 21500之後，PMI-Taipei Taiwan Chapter(PMI-TW) 再接再厲、號召志工陸續把 ISO 21504、ISO 21505 轉化為國家標準 CNS 21504、CNS 21505，2018年的目標則是將 ISO 21503、ISO 21508、ISO 21511也進行轉化，以期能加速接軌國際的腳步。

這一系列專案管理國際標準轉化為國家標準，緣起於 PMI-TW「有感於企業對於專案管理領域並無公定標準可遵循，於是決定將 ISO 21500（專案管理指引）進一步轉化為專案管理指引之國家標準，也就是 CNS 21500，期望能有效促進國內專案管理發展。」的初衷，之後，為了讓台灣產業及企業環境在實際執行專案上有更好的依循標準，健全台灣專案管理生態，所以持續進行 ISO 21504、ISO 21505 等後續國際標準轉化為國家標準的翻譯。

目前台灣將 ISO 的專案管理標準轉化成國家標準的速度在國際間也是最快的。這些標準能帶給台灣的好處有三：

1. 可為企業導入專案管理制度的依據 (CNS 21500、CNS 21504、CNS 21505、ISO 21503)

2. 成為業界之間的共同語言

3. 讓政府與企業能跟國際接軌

所有 ISO 專案管理相關標準與台灣轉化列表

CNS 21505是繼2016年CNS 21500(專案管理指引)與2017年CNS 21504(專案組合管理指引)陸續出爐後，第三個發布的專案管理標準。這些標準的訂定能在不到2年的時間以如此驚人的速度成長，有大部分是仰賴於一群高效率的PMP志工團隊。

針對本次CNS 21505發布，更值得關注的一點是，國際標準化組織 (International Organization for Standardization; ISO) 在2017/5/4頒布ISO 21504後，台灣與瑞典是全球唯二(另一為瑞典)將其列入國家標準者。

目前仍正轉化中的專案管理國家標準尚有ISO 21503(專案集管理指引)及ISO 21508(實獲值管理指引)及ISO 21511 (專案/專案集管理的WBS)，中文名稱尚未定案。預計2018年底就能頒布，未來將可繼續期待台灣企業在專案管理領域的高度發展。

最終以標檢局局長 劉明忠的話來說明此專案的成功關鍵：「一個在專案管理領域讓產官學「動起來」的超級專案–制定專案管理的國家標準，成功地整合國內的專案管理組織團體，我覺得這個團隊很棒的作為是，每個人都能發揮自己的優點，舉例來說，PMI-TW 國際標準在地化委員會主委周龍鴻號召百餘位志工以敏捷專案管理完成CNS草案；PMI-TW 理事長陳威良則是以提案單位 PMI-TW 的身分跟標檢局做良善溝通；而中華專案管理學會許秀影理事長則用嚴謹的角度，做最後專業上的把關。每一個專案管理團隊都發揮自身的特長，不分你我、無私地全部整合在一起，單純只是為了一件事—把國家標準做好，實在是專案管理界最好的表率。

這是一個偉大的里程碑，國家標準的影響力可在政府、民間、企業之間展開，絕對是專案管理更上一層樓的火車頭。
